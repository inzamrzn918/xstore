# Mobile App - SecurePrintShare

Expo React Native mobile app with Bun package manager.

## Quick Start

```bash
# Install dependencies
bun install

# Update API_URL in App.js with your computer's IP
# Example: const API_URL = 'http://192.168.1.100:8000';

# Start Expo
bun run start

# Scan QR code with Expo Go app
```

For full documentation, see the [main README](../README.md).
